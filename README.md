#### Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     dogehouse_cr:
       github: willdoescode/dogehouse_cr
   ```

2. Run `shards install`

#### Usage

```crystal
require "dogehouse_cr"
```
