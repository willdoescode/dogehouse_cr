# dogehouse_cr

## The fastest way to make the fastest dogehouse bots.

### [Docs](http://docs.will.tools/docs/)
### [Examples](examples/)

#### Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     dogehouse_cr:
       github: willdoescode/dogehouse_cr
   ```

2. Run `shards install`

#### Usage

```crystal
require "dogehouse_cr"
```
