# dogehouse_cr

A dogehouse wrapper for crystal

## [Docs](http://docs.will.tools/docs/)

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     dogehouse_cr:
       github: willdoescode/dogehouse_cr
   ```

2. Run `shards install`
